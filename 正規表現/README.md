## 正規表現Tips

```
# 数字n桁
^\d{7}$

# 数字ハイフン付きn桁
^\d{3}-\d{4}$

# 電話番号
^0\d{2,3}-\d{1,4}-\d{4}$

# 携帯電話番号
^(070|080|090)-\d{4}-\d{4}$

# メールアドレス
^[a-zA-Z0-9-_\.]+@[a-zA-Z0-9-_\.]+$

# 日付
\d+/\d+/\d+

# 行頭空白
^\s+

# 末尾空白
\s+$

# abcの後にdefがない場合のabcに一致
abc(?!def)

# foo/barを含むものに一致
(?=.*foo)(?=.*bar)
```
