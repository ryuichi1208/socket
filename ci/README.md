## 用語

- CI
  - Continuous Integration
  - コミット/テスト/結合を自動化する為のツール
    - Jenkins
    - CircleCI
    - TravisCI
    - CodeBuild(AWS)
- CD
  - Continuous Delivery
  - 継続的デリバリー
  - リリースプロセス全体を自動化
    - CodeDeploy(AWS)

## Kubernetes利用に向けたCIパイプライン構築ハンズオン

リリースの短期間化

Dev => コーディング/ビルド/テスト/リリース

Ops => デプロイ/運用/監視
